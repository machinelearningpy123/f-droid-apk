# Research Report

This F-Droid refresh is based on the official F-Droid homepage, the official package page for `org.fdroid.fdroid`, and the stable APK headers checked on June 5, 2026. The homepage presents F-Droid as a trusted home for free and open source mobile apps and links directly to `F-Droid.apk`. The official package page makes an important distinction that many low-quality download pages skip: the direct APK route is available, but F-Droid says that installing that way is less secure and does not provide update notifications, which is why it recommends installing the F-Droid client and using that. The stable homepage APK currently aligns with version `1.23.2`, a file around `11.85 MiB`, Android `6.0+`, and a last-modified date of `February 19, 2026`. The same package page also lists newer beta builds up to `2.0-alpha9`, which require Android `7.0+` and are larger at around `15 MiB`. The package page also exposes trust signals that matter for human readers: package name `org.fdroid.fdroid`, build metadata, reproducibility links, and the note that the APK is built and signed by F-Droid.
